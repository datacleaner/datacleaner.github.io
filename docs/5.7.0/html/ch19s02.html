Building DataCleaner

Get the source code for DataCleaner from GitHub: 

			> git clone https://github.com/datacleaner/DataCleaner.git DataCleaner

Build the projects: 

			> cd DataCleaner
			> mvn clean install

Run DataCleaner

			> cd desktop/ui/target
			> java -jar DataCleaner-desktop-ui-[version].jar
