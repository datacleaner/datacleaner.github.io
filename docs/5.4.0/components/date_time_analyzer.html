Date/time analyzer

Records a variety of interesting measures for date or time based data. Which are the highest/lowest values? How is the year distribution of dates? Are there null values?

Analyzer Concurrent

Result metrics (DateAndTimeAnalyzerResult)

  • Row count

    Parameterized by column

  • Null count

    Parameterized by column

  • Highest date

    The highest date value for the given column. The value is measured in number of days since 1970-01-01.

    Parameterized by column

  • Lowest date

    The lowest date value for the given column. The value is measured in number of days since 1970-01-01.

    Parameterized by column

  • Mean

    The mean value for the given column. The value is measured in number of days since 1970-01-01.

    Parameterized by column

  • Median

    The median value for the given column. The value is measured in number of days since 1970-01-01.

    Parameterized by column

  • 25th percentile

    The 25th percentile value for the given column. The value is measured in number of days since 1970-01-01.

    Parameterized by column

  • 75th percentile

    The 75th percentile value for the given column. The value is measured in number of days since 1970-01-01.

    Parameterized by column

  • Kurtosis

    Parameterized by column

  • Skewness

    Parameterized by column

Configured properties

  • Columns

    List of InputColumn<Date> Required

  • Descriptive statistics

    Gather so-called descriptive statistics, including median, skewness, kurtosis and percentiles, which have a larger memory-footprint.

    boolean Required
